# Personal

General long-term personal preferences, routines, and non-sensitive assistant context.

## What FREEOS should store here

Keep approved project knowledge, working notes, references, and user-provided artifacts related to Personal.

## Example notes

- Decisions and current priorities
- Approved project context
- Useful references and non-sensitive planning notes

All files in this folder stay local. FREEOS does not scan or save private files automatically.
